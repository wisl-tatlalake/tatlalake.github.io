# Site images

These are **placeholder** images (solid gradients). Replace them with real
cottage photos, keeping the same file names so no page edits are needed:

| File | Used for | Suggested size |
| ---- | -------- | -------------- |
| `hero.jpg` | Full-width homepage hero banner | ~1600×900, landscape |
| `gallery-1.jpg` … `gallery-6.jpg` | Home preview + `/gallery/` grid | ~1000×750 each |

To add more gallery photos, drop them here (e.g. `gallery-7.jpg`) and add a
matching entry to the `gallery:` list in `gallery.md`.

Tips:
- Use landscape JPEGs, optimized to a reasonable size (keep each under ~500 KB).
- The first gallery image also works well as a social-share (`og:image`) —
  it is picked up automatically from the page.
